三，Typora+PicGo+Github保证博客中图片显示

2023-11-08 评论

为了方便写作，我们可以采用Typora+PicGo+Github的方案来搭建我们的博客写作环境。

  • Typora：它是一款轻便简洁的Markdown编辑器，支持即时渲染技术，即所写立刻所见，少了排版的时间，专注于文章内容的编辑。
  • PicGo:它是一个用于快速上传图片并获取图片 URL 链接的工具，支持多种图库。
  • Github:代码托管工具，除了代码，还可以用作图片存储。

在我们的博客写作中，可以达成这样的需求：

  • 使用Markdown编辑博客
  • 一篇博客发布多个平台
  • 图片一次上传，处处引用

操作步骤：

  1. 安装好Typora和PicGo软件，并准备一个GitHub账号
  2. 登录Github并创建一个公开的仓库，需要注意仓库必须是公开的，否则无法预览。
  3. 设置token
  • 点击右上角头像-settings，然后点击developer settings，点击personal access tokens。
  • 点击tokens (classic)，点击create new token(classic)
  • 设置权限
  • 复制token
  1. PicGo配置Github图床
  • 点击左边图床设置，选择GitHub图床，具体配置如下
  • 设定仓库名，填写：GitHub名/库名
  • 分支，默认填main
  • 设定Token，刚才保存的token令牌
  • 指定存储路径，默认填img/

自定义域名为https://cdn.jsdelivr.net/gh/你的github用户名/你的仓库名@分支（一般为main）/
